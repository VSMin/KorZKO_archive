# Корейцы Приуралья: 90 лет истории — прототип портала

Интерактивный прототип публичного портала цифрового архива (кластер «Память Приуралья», ЗКО филиал АКК).

`index.html` — автономный файл: все стили, скрипты и шрифтовые ссылки собраны внутрь. Работает на любом статическом хостинге.

## Публикация на GitHub Pages

1. Создайте репозиторий (например, `priuralye-archive`).
2. Загрузите содержимое этой папки в корень репозитория (`index.html`, `README.md`).
3. Settings → Pages → Source: `Deploy from a branch`, ветка `main`, папка `/ (root)`.
4. Через минуту сайт будет доступен по адресу `https://<ваш-логин>.github.io/priuralye-archive/`.

Альтернативы: Netlify Drop (перетащить папку на netlify.com/drop), Vercel, Cloudflare Pages.

## Примечания

- Карта расселения использует открытые тайлы CARTO через MapLibre GL — нужен интернет у посетителя.
- Плейсхолдеры фотографий заполняются в исходном проекте; в собранной версии зашиты текущие изображения.
- Это дизайн-прототип, не production-система из ТЗ (NestJS/PostgreSQL): он предназначен для демонстрации и сбора обратной связи.
